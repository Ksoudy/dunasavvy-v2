# DunaSavvy — Static Site (`/docs`)

This folder is the pre-built React landing page, ready to serve from GitHub Pages.

- **Entry point:** `index.html`
- **Assets:** `static/css/main.*.css`, `static/js/main.*.js`
- `.nojekyll` → tells GitHub Pages to skip Jekyll processing
- `CNAME` → custom domain (`dunasavvy.com`)

See `../DEPLOY.md` for the full deploy playbook and how to rebuild this folder.
